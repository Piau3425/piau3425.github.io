前綴和

前綴和是對於數列的一種預處理
經過 O(n) 初始化過後可以在 O(1) 就取得 [a, b] 的區間和

一維前綴和

要點

功能

  • query：查詢某範圍的區間和

觀念

一維前綴和就是將第 N 項的數字改為 N 之前的所有總和
之後每次進行查詢我們就只需要輸出第 b 項數字減去第 a - 1 項數字

為了避免 a - 1 會減到負數
我們需要將索引改成 1-based，並將第 0 項數字設為 0
